## 🌌 Overview

DevSphere is a next-generation developer ecosystem that combines collaboration, AI intelligence, and social networking into one unified platform.

> 💡 Designed to go beyond GitHub, GitLab, and traditional dev tools.

---

## ✨ Features

🚀 Real-time collaborative coding
🤖 AI-powered code assistant
📄 AI resume analyzer
🌐 Developer social network
⚡ Online code execution
🔗 GitHub profile integration

---

## 🧠 Vision

To build the **operating system for developers** — where coding, collaboration, and growth happen in one place.

```bash
   ↓
MongoDB + Socket.io + AI APIs
```

---

## 🚀 Getting Started

```bash
# Clone repo
git clone https://github.com/your-username/devsphere.git

# Backend
cd server
npm install
npm start

# Frontend
cd client
npm install
npm start
```
